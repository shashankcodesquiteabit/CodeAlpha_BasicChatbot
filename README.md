# CodeAlpha_BasicChatbot

A simple rule-based chatbot built in Python as part of the **CodeAlpha Python Programming Internship**.

## 📌 Description
This chatbot responds to common user inputs (greetings, farewells, small talk) using predefined rules and `if-elif` logic. No AI or external APIs are used — it's a fundamentals project demonstrating core Python concepts.

## 🎯 Key Concepts Used
- Conditional logic (`if-elif`)
- Functions
- Loops (`while`)
- Input/output handling
- String methods (`.lower()`, `.strip()`, `.split()`)

## 🚀 How to Run
1. Make sure Python 3 is installed.
2. Clone this repository:
   ```
   git clone https://github.com/YOUR_USERNAME/CodeAlpha_BasicChatbot.git
   cd CodeAlpha_BasicChatbot
   ```
3. Run the script:
   ```
   python chatbot.py
   ```
4. Start chatting! Type things like:
   - `hello`
   - `how are you`
   - `tell me a joke`
   - `my name is Shanky`
   - `bye` (to exit)

## 💬 Example Conversation
```
You: hello
Bot: Hi there!
You: how are you
Bot: I'm fine, thanks! How about you?
You: tell me a joke
Bot: Why do programmers prefer dark mode? Because light attracts bugs!
You: bye
Bot: Goodbye! Have a great day!
```
